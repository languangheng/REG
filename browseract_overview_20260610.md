# BrowserAct 项目概览

## 目标
了解 BrowserAct 项目的基本信息。

## 关键信息
- **项目定位**：专为 AI Agent 打造的反检测浏览器自动化 CLI 工具
- **GitHub**：https://github.com/browser-act/skills（~1.4K Star）
- **核心差异化**：三层反拦截体系（环境层/执行层/人机交互层）

## 三层突破体系
1. **环境层**：stealth 反检测 + 指纹伪装 + 动态代理 + Session/Cookie/Profile 隔离
2. **执行层**：solve-captcha 自动解验证码 + stealth-extract 提取受保护内容
3. **人机交互层**：remote-assist 远程接管，人工完成后 Agent 继续原会话

## 同类项目
- Browser Use（github.com/browser-use/browser-use）：基于 LangChain，验证码能力较弱

## 结论
BrowserAct 的核心价值在于反检测 + 验证码处理 + 人机协作的完整链路，适合需要绕过网站拦截的 AI Agent 自动化场景。
