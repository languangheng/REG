## 🚀 如何使用

### 方法1: 直接打开（最简单）
1. 打开 `theory_origin` 文件夹
2. 双击 `theory_origin.html`
3. 自动在浏览器中打开

### 方法2: 使用启动脚本（Windows）
1. 双击 `启动网页.bat`
2. 选择 "1" 直接用浏览器打开，或选择 "2" 启动本地服务器

### 方法3: 命令行启动服务器
```bash
cd theory_origin
python -m http.server 8000
```
然后访问: http://localhost:8000/theory_origin.html
