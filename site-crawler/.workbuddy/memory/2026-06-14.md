# 2026-06-14 工作日志

## 引导式向导实现（Task #10, #6, #7, #8）

### 新增文件
- `crawler/guided_wizard.py` — 引导向导核心类
  - 数据模型: GuidedLevel, PaginationInfo, GuidedSession
  - URL 泛化: `generalize_path()` — 将数字替换为 `\d+`，扩展名点号转义
  - 翻页检测: `detect_pagination()` — 支持 5 种格式 (query_param, path_page, maccms, suffix_page, query_other)
  - GuidedWizard 类: start/get_status/cancel/start_pagination_detect/end_pagination_detect/add_path/update_path_pattern/detect_resource/complete
  - 配置生成: `generate_site_config()` 从 levels + pagination → sites.json 格式

- `templates/wizard_guided.html` — 引导向导前端页面
  - 左面板: 启动区 + 操作区(翻页检测/路径管理/完成) + 层级列表 + 日志
  - 右面板: 配置预览(实时构建 JSON)
  - 编辑弹窗: 双击 path_pattern 可编辑

### 修改文件
- `site_manager.py` — 新增 10 个 `/api/guided/*` API 端点
  - /wizard_guided (GET) — 前端页面
  - /api/guided/start (POST) — 开始引导
  - /api/guided/status (GET) — 获取状态
  - /api/guided/pagination/start (POST) — 开始翻页检测
  - /api/guided/pagination/end (POST) — 结束翻页检测
  - /api/guided/path/add (POST) — 添加路径
  - /api/guided/path/update (POST) — 编辑路径模式
  - /api/guided/resource/detect (POST) — 检测资源
  - /api/guided/complete (POST) — 完成保存
  - /api/guided/cancel (POST) — 取消引导

- `templates/index.html` — 入口按钮修改
  - "🔍 自动分析" + "🎬 HAR 录制" → "➕ 添加网站" (指向引导向导)
  - openHarWizard() → openGuidedWizard()
  - 卡片中"添加分组"按钮也改为指向引导向导
