# OVERLAY_JS 中文化 → 英文化

## 目标
将 `config_wizard.py` 的 `OVERLAY_JS` 中所有中文 UI 文本和注释替换为英文，解决 `browser-act eval` 返回中文时的 GBK 编码乱码问题。

## 方案
用 Python 脚本批量替换，分三批：
1. 第一批：主要 UI 文本（之前的 edit 部分成功）
2. 第二批：剩余 79 个中文字符串（含注释）
3. 第三批：最后 11 个残留（短字/助词类）

## 结果
- ✅ 所有中文字符已从 OVERLAY_JS 中清除
- ✅ 测试注入成功：cw-panel FOUND, __crawler_wizard = true
- ✅ 用户确认浏览器中面板可见

## 修改文件
- `crawler/config_wizard.py` — OVERLAY_JS 全部英文化

## 备注
- 中文乱码根因：`browser-act` (Node.js) 通过 PowerShell pipe 接收 stdin 时，返回结果的编码转换在 Windows GBK 环境下出错
- 解决方案：OVERLAY_JS 中不使用任何非 ASCII 字符
